The Internet

The Internet is considered the largest network in the world. It connects billions of devices globally, including computers, smartphones, servers, and various other digital devices. This vast network enables communication and data exchange across different continents and countries.
These devices communicate with each other to share information and resources. This network of devices includes personal computers, mobile phones, servers, routers, and many other types of technology. The connections among these devices form a complex web that facilitates the flow of data across the globe.

WWW-The World Wide Web

It is a network of interconnected computers managing information across multiple locations. The Method of linking and navigating between documents or sections is called Hypertext. In a web page, HTML is the language used to format and create web documents, including text, images, forms, and embedded files, and CSS is used with HTML to apply styles and formatting to web pages.

Types of Web Pages

  • Static pages
    • Built using HTML, CSS and JavaScript.
    • Not updated very often.
    • Do not have a database connected with the page.
  • Dynamic pages
    • HTML is generated at run time based on the request.
    • Often CSS and JavaScript is static.
    • Have a database connected with the page where content is stored.

HyperText Markup Language

  • Most basic building block of the Web.
  • Gives structure and meaning to a web page.
  • “Hypertext” links web pages to each other.
  • Links may be to the same or a different website.
  • Latest standard is HTML5.

CSS - Cascading Stylesheets

  • Used to describe the presentation of a web page.
  • Provides design consistency.
  • Historically had versioning like CSS1, CSS2, CSS3.
