Overview

CSS is a stylesheet language that is used to provide design to web pages. CSS rulesets style the HTML elements as defined by the declarations written in the rulesets. A CSS ruleset (normally called just rule) targets an HTML element using a selector and defines its appearance

Defining Fonts, Colours, and Sizes.

Fonts

CSS is an effective way to create consistency of fonts across your website. Through CSS, you can define:

  • font-family
  • font-size
  • font-weight
  • font-style
  • font-height
  • font-color
  • external font files

Colours

There are seven ways to define colours in CSS:

  • 6-digit hex
  • 3-digit hex
  • named colours
  • rgb method
  • rgba method
  • hsl method
  • hsla method

Sizes

You can define sizes in CSS in any of the following units:

  • Relative
  • Absolute
  • Viewport
