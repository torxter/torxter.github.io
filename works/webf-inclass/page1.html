What is HTML?

HTML is the standard markup language for creating Web pages.

HTML Page Structure

  1. DOCTYPE html document type declaration
  2. html to contain all page code
    1. head
    2. body to contain page content

Overview

  • The most basic building block of the Web.
  • Gives structure and meaning to a web page.
  • "Hypertext" links web pages to each other. Links may be to the same or a different website.
  • HTML markup is built using tags.
  • The latest standard is HTML5

Basic sections of a website

HTML5 introduced semantic elements and containers for the purpose of accessiblity and better site organization. Here are some basic sections on a website:

  • header to contain the main heading of the page
  • nav is the main site's navigation bar
  • main to contain the main content of the page
  • section to section the page and group content
  • aside to create a sidebar for the page
  • footer to contain the copyright and contact information

HTML Syntax

  • An HTML element is identified using tags.
  • A tag consists of the name of the element enclosed in ‹ and ›.
  • HTML is case insensitive.
  • To write a paragraph on a web page, use opening and closing p tags to enclose the text.

Basic HTML Syntax

  • paragraphs
  • headings
  • hyperlinks
  • images
  • lists; ordered and unordered
  • text markup elements
